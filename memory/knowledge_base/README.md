# 知识库使用说明

这个目录存放领域相关的知识资料。

## 目录结构

```
knowledge_base/
├── domain_data/      # 领域资料
│   ├── 行业报告
│   ├── 核心数据
│   └── 专家观点
├── case_studies/     # 案例库
│   ├── 爆款文章分析
│   └── 经典案例
└── assets/           # 素材库
    ├── 金句合集
    └── 数据图表
```

## 添加内容

请将你的领域资料放入对应目录，支持格式：
- Markdown (.md)
- JSON (.json)
- 文本 (.txt)

## 使用方式

创作时会自动读取相关知识库内容作为参考。
